# Setup Instructions

## Prerequisites

Node.js Version 16 or higher:

```bash
node --version
```

## Installation

```bash
npm install
```

You should see:

```
added 428 packages in 5s
```

(Ignore any watchman warnings - they don't affect functionality)

## Running Tests

```bash
npm test
```

## Project Structure

- `calculator.js` - Calculator functions (already implemented)
- `calculator.test.js` - Test file (with TODOs for you to complete)
