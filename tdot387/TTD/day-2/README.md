# Setup Instructions

## Installation

```bash
npm install
```

## Running Tests

```bash
# Run tests once
npm test

# Run tests in watch mode
npm test:watch
```

## Project Structure

- `stringUtils.js` - String utility functions (with intentional bugs)
- `stringUtils.test.js` - Tests to refactor (vague assertions that let bugs through)
