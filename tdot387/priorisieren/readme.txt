Übung 1 (Eingerückt sind die Teilaufgaben)

Aufteilung eines Projekts für eine persönliche Portfolio-Website:

- Projekt anlegen
- Header erstellen
    - Logo erstellen    
    - Menu erstellen
- Über mich Sektion
    - Persönliches Bild auswählen
    - Text über eigenen Werdegang verfassen
- Eingene Projekte präsentieren
    - Bestes Projekt auswählen  
        - Github-Link raussuchen
        - Website mit Vorschau des Projekts verlinken
        - Beschreibungstext (inkl verwendeter Technologien)
    - Weitere Projekte auswählen, nach gleichem Schema verfahren
- Kontaktbereich
    - Kontaktformular erstellen
    - Mailadresse und Social Media angeben
- Footer
    - Impressum
    - Datenschutz
